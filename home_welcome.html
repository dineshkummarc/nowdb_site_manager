Tentang Site Manager

Site Manager merupakan CMS yang di bangun dengan html & javascript yang memanfaatkan NowDB sebagai data storage, berikut ini daftar fitur yang dimiliki Site Manager:

  • Pengelolaan Artikel
    • Input Artikel
    • Edit Artikel
    • Hapus Artikel
    • Input Konten Artikel
    • Edit Konten Artikel
    • Hapus Konten Artikel
    • Input Kategori Artikel
    • Tautkan Kategori
    • Hapus Tautan Kategori
    • Input Label
    • Hapus Label
